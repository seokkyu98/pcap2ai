문의 전 확인해 주세요

  • 변환이 시작되지 않아요 — 무료 서버는 절전 상태에서 깨어나는 데 최대 90초가 걸립니다. 화면 상단의 서버 상태 표시가 "온라인"이 된 뒤 다시 시도해 주세요.
  • "다른 변환 작업이 진행 중" 메시지가 나와요 — 서버가 동시에 한 건의 변환만 처리하기 때문입니다. 몇 분 뒤 다시 시도하면 됩니다.
  • 파일이 100MB를 넘어요 — Wireshark의 editcap -c 100000 in.pcap out.pcap 명령으로 캡처를 분할한 뒤 나눠서 변환해 주세요.

오류 제보 시 포함하면 좋은 정보

  1. 사용한 브라우저와 버전 (예: Chrome 126)
  2. 업로드한 파일의 대략적인 크기와 형식 (.pcap / .pcapng)
  3. 선택한 변환 모드 (요약 / 상세)
  4. 화면의 진행 로그에 표시된 마지막 메시지
